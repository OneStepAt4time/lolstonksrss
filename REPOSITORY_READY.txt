================================================================================
                           GITHUB UPLOAD METHODS
================================================================================

Method 1 - GitHub CLI (Recommended):
  $ gh auth login
  $ gh repo create lolstonksrss --public --source=. --remote=origin --push

Method 2 - Web + Git CLI:
  1. Create repo at https://github.com/new
  2. $ git remote add origin https://github.com/USERNAME/lolstonksrss.git
  3. $ git push -u origin master

Method 3 - SSH:
  $ git remote add origin git@github.com:USERNAME/lolstonksrss.git
  $ git push -u origin master
